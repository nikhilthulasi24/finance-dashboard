## 📝 DEPLOYMENT TROUBLESHOOTING

### If Deployment Fails
1. Check **Actions tab** for error logs
2. Common issues:
   - Build fails → Usually Node version mismatch (uses 18, should work)
   - Deploy fails → Check GitHub Pages settings in repository
3. Fix and push again: Changes auto-trigger redeployment

### If GitHub Pages Not Showing
1. Go to repository **Settings**
2. Scroll to **GitHub Pages** section
3. Verify: Source = `Deploy from a branch`
4. Branch = `gh-pages` (auto-created by workflow)
5. Folder = `/ (root)`
6. Wait 2-3 minutes for DNS propagation

### Manual Deployment Option
If automated fails, you can manually deploy:
```bash
# Build locally
npm run build

# Deploy using GitHub CLI
gh pages deploy dist --branch gh-pages
```
